## Why

<CardGrid>
  <Card title="Any Backend" icon="puzzle">
    Built on a simple adapter interface. Ships with Hyperserve support; bring your own Mux, Bunny, api.video, or custom backend.
  </Card>
  <Card title="Cross-Platform" icon="laptop">
    Same core logic for React (web) and React Native. Platform-specific UI packages with identical APIs.
  </Card>
  <Card title="Composable UI" icon="setting">
    Pre-built compound components you can assemble, rearrange, or replace. Or go fully headless and build your own.
  </Card>
  <Card title="Production Ready" icon="approve-check">
    Concurrency control, abort/retry, progress tracking, client-side validation, post-upload status monitoring.
  </Card>
</CardGrid>
